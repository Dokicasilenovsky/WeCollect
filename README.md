# WeCollect

_WeCollect is an Android application designed to help users compare product prices across different stores through a community-driven database. Instead of relying on official retailer data, the application allows users to contribute by adding and updating product prices, creating a continuously growing and up-to-date collection of information._

_The project was developed as a team project with the goal of creating a practical mobile application while gaining experience in Android development, Firebase services, and backend integration._

---

## Features

* User registration and authentication
* Product management
* Barcode scanning for faster product entry
* Product search by name
* Store-based price comparison
* Community-driven price updates
* Cloud data synchronization using Firebase
* Companion web application for browsing the product database

---

## Architecture

The application follows a modular Android architecture where the client communicates directly with Firebase services.

### Main Components

* Authentication module
* Product management
* Barcode scanning
* Search functionality
* Firebase Realtime Database integration
* User management
* Web database viewer

Firebase is responsible for storing application data and handling user authentication, allowing real-time synchronization between all users.
